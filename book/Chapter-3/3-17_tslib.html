3.17 tslib

3.17 tslib

官网：http://www.tslib.org/
源码：tslib-1.22.tar.gz

Step 1：安装依赖工具
sudo apt-get install autoconf
sudo apt-get install automake
sudo apt-get install libtool
Step 2：解压源码
tar -xvzf tslib-1.22.tar.gz
cd tslib-1.22/
Step 3：创建安装目录
mkdir arm_tslib
Step 4：配置
./configure --host=aarch64-linux-gnu ac_cv_func_malloc_0_nonnull=yes --cache-file=arm-linux.cache -prefix=/home/xxx/share/3rdparty/tslib-1.22/arm_tslib
Step 5：编译安装
make
make install
Step 6：板端部署

将编译安装输出文件夹放到板端根文件系统下。

  • 将 tslib 的 lib 拷贝到板端根文件系统 /usr/lib 下；
  • 将 tslib 的 etc/ts.conf 拷贝到板端根文件系统 /etc 下；
  • 将 tslib 的 bin 拷贝到板端根文件系统 /usr/local/tslib 下；

嵌入式板端 /etc/profile 配置环境变量，如下 

export TSLIB_ROOT=/usr/local/tslib 
export TSLIB_CONSOLEDEVICE=none 
export TSLIB_FBDEVICE=/dev/fb0 
export TSLIB_TSDEVICE=/dev/input/event1 
export TSLIB_CONFFILE=/etc/ts.conf 
export TSLIB_PLUGINDIR=/usr/lib/ts 
export TSLIB_CALIBFILE=/etc/pointercal 
export LD_PRELOAD=/usr/lib/libts.so

注意：TSLIB_TSDEVICE 是触摸屏驱动的设备节点，根据实际修改。

Step 7：触摸校准

使用编译生成的 ts_calibrate 工具校准触摸，将校准参数保存在 /etc/pointercal 文件。

Step 8：测试体验

可以用编译生成的 bin 文件夹下可执行文件测试体验，如 

ts_test
