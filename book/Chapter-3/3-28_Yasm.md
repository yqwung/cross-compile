### 3.28 Yasm

下载：http://yasm.tortall.net/Download.html <br>
源码：yasm-1.3.0.tar.gz   <br>

#### ubuntu 系统编译

##### Step 1：解压源码

```
tar -xvzf yasm-1.3.0.tar.gz
cd yasm-1.3.0/
```

##### Step 2：配置

```
./configure
```

##### Step 3：编译，安装

```
make
make install
```
