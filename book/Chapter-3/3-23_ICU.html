3.23 ICU

3.23 ICU

官网：https://icu.unicode.org/
下载：https://github.com/unicode-org/icu/releases/tag/release-58-3
源码：icu4c-58_3-src.tgz

Step 1：解压源码
tar -xvzf icu4c-58_3-src.tgz
cd icu/
Step 2：ubuntu编译

1）新建安装目录

mkdir ubuntu_icu

2）编译

cd ubuntu_icu/
../source/runConfigureICU Linux/gcc
make

3）可能会有如下编译报错

报错1： 
../../source/i18n/digitlst.cpp:67:13: fatal error: xlocale.h: No such file or directory
 #   include <xlocale.h>
             ^~~~~~~~~~~
compilation terminated.

解决办法：

sudo ln -s /usr/include/locale.h /usr/include/xlocale.h

参考：https://blog.csdn.net/u010835747/article/details/120377923

4）继续编译

make
Step 3：交叉编译配置配置

1）新建安装目录

mkdir arm_icu

2）交叉编译配置

cd source/

./configure --disable-samples --disable-tests  \
--with-cross-build=/home/xxx/share/3rdparty/icu/ubuntu_icu  \
--prefix=/home/xxx/share/3rdparty/icu/arm_icu  \
--host=aarch64-linux-gnu CC=aarch64-linux-gnu-gcc

3）编译，安装

make
make install

可能会有如下编译报错

报错1： 
aarch64-linux-gnu-g++ -O2 -W -Wall -pedantic -Wpointer-arith -Wwrite-strings -Wno-long-long --std=c++0x      -shared -Wl,-Bsymbolic -Wl,-soname -Wl,libiculx.so.58 -o ../lib/libiculx.so.58.3 ParagraphLayout.o RunArrays.o LXUtilities.o playout.o plruns.o -L../lib -licuuc -L../stubdata -licudata -I/usr/include/icu-le-hb -I/usr/include/harfbuzz -I/usr/include/glib-2.0 -I/usr/lib/x86_64-linux-gnu/glib-2.0/include -L../lib -licu-le-hb -lpthread -ldl -lm 
/home/xi/opt/arm-linux/gcc-linaro-7.5.0-2019.12-x86_64_aarch64-linux-gnu/bin/../lib/gcc/aarch64-linux-gnu/7.5.0/../../../../aarch64-linux-gnu/bin/ld: cannot find -licu-le-hb
collect2: error: ld returned 1 exit status
Makefile:165: recipe for target '../lib/libiculx.so.58.3' failed
make[1]: *** [../lib/libiculx.so.58.3] Error 1
make[1]: Leaving directory '/home/xxx/share/3rdparty/icu/source/layoutex'
Makefile:143: recipe for target 'all-recursive' failed
make: *** [all-recursive] Error 2

可以不要 icu-le-hb ，修改编译配置，添加如下项：

--enable-layoutex=no

整体配置如下：

./configure --disable-samples --disable-tests --enable-layoutex=no  \
--with-cross-build=/home/xxx/share/3rdparty/icu/ubuntu_icu  \
--prefix=/home/xxx/share/3rdparty/icu/arm_icu  \
--host=aarch64-linux-gnu CC=aarch64-linux-gnu-gcc

4）重新配置、编译、安装
